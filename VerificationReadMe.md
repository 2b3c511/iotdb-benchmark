# Related BenchmarkMode
Now you can use generateDataMode to generate dataSet(which will generated by benchmark), and it will be store at FILE_PATH(Notice: FILE_PATH should be an empty directory).

Then you can use verificationMode to insert dataSet and query dataSet.

Because of time, now it only supported IoTDB 0.12

# Process
1. Firstly, you can use generateDataMode to generate dataSet(FILE_PATH=data/test):

